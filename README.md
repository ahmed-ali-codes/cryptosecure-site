# CryptoSecure – 3D Animated Crypto Platform Landing Page

[![Made with Three.js](https://img.shields.io/badge/Three.js-r128-green?logo=three.js)](https://threejs.org/)
[![HTML5](https://img.shields.io/badge/HTML5-Semantic-orange)](https://validator.w3.org/)
[![CSS3](https://img.shields.io/badge/CSS3-Glassmorphism-blue)](https://www.w3.org/Style/CSS/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

**CryptoSecure** is a premium, fully responsive landing page for a professional digital asset management platform. It features immersive Three.js 3D particle fields, wireframe geometry animations, glassmorphism UI, scroll-triggered reveals, and a dark-mode design system — ideal as a portfolio showcase or startup landing page.

---

## ✨ Key Features

- **3D Particle Universe** — Hero section with 1,500+ interactive particles, wireframe torus knot, icosahedron & octahedron
- **Multi-Scene 3D** — Three separate Three.js scenes (hero, stats, CTA) for visual depth
- **Mouse-Reactive** — Particles and geometry respond to cursor movement in real time
- **Scroll Animations** — Intersection Observer–powered fade-in and slide-in animations
- **Animated Counters** — Stats numbers count up with eased cubic timing
- **Cursor Glow Effect** — Subtle ambient glow that follows the cursor (desktop)
- **Glassmorphism Cards** — Feature cards with dynamic glow tracking on hover
- **Preloader** — Branded dual-ring loading animation
- **Fully Responsive** — Mobile-first design with hamburger nav and adaptive grids
- **WhatsApp Integration** — Floating contact button with custom message
- **Scroll-to-Top Button** — Appears on scroll with smooth animation
- **Professional Copy** — Regulatory-compliant language, no misleading claims
- **Risk Disclosure** — Full legal disclaimer footer section
- **Crypto Ticker Bar** — Scrolling animated marquee of asset symbols

## 🚀 Installation & Setup

This is a static front-end project — no build steps required.

### 1. Clone the repository

```bash
git clone https://github.com/ahmed-ali-codes/cryptosecure-site.git
cd cryptosecure-site
```

### 2. Open in browser

Simply open `index.html` in any modern browser, or use a local server:

```bash
npx serve .
```
